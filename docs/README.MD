# Projekt Quiz

## Opis

Gra quizowa dla wielu graczy, umożliwiająca odpowiadanie na pytania z różnych kategorii.

## Instrukcja użytkownika

- Uruchom grę
- Wybierz kategorię pytań
- Odpowiedz na pytania
- Uzyskaj wynik
